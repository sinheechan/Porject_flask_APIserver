## Object

Streamlit은 파이썬 스크립트로 데이터를 시각화하고 웹 애플리케이션을 생성할 수 있다.

본 프로젝트에서는 해당 도구를 활용하여 웹 서비스 구축 모델 생성을 목표로 한다.

1. Mnist 손글씨 분류 및 시각화 웹 애플리케이션
2. 비트코인의 가격 및 거래량 시각화 웹 애플리케이션
